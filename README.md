# SistemaDePrendido
Este sistema de prendido(super básico) permite prender un relé a distancia usando un ESP8266 por medio de un celular con Blynk.

Se conecta el hardware a una red Wi-Fi y a una cuenta de Blynk. El código no está pulido y puede haber varios errores si no se usa con cuidado. En mi caso, estoy utilizando esto para encender una PC a distancia (ya que la misma no tiene ningún protocolo de encendido remoto).

Para lograrlo, se conecta la salida del relé al mismo par de cables del botón de encendido de la PC. Lo bueno de este sistema es que el relé no interfiere con ningún sistema de la computadora.
